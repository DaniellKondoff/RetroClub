# Ръководство за добавяне на събития в Contentful

Това ръководство обяснява как да добавите или редактирате събитие в системата за управление на съдържание (Contentful) и кое поле какво показва на уебсайта.

---

## Как да създадете ново събитие

1. Влезте в [Contentful](https://app.contentful.com)
2. От менюто вляво изберете **Content**
3. Натиснете бутона **Add entry** (горе вдясно)
4. Изберете тип съдържание **Event**
5. Попълнете полетата (описани по-долу)
6. Натиснете **Publish** за да публикувате събитието на сайта

---

## Полета

### 1. Date (Дата)

| | |
|---|---|
| **Поле в Contentful** | `Date` |
| **Какво да въведете** | Датата на събитието във формат `ГГГГ-ММ-ДД` |
| **Пример** | `2025-06-14` (за 14 юни 2025 г.) |

**Къде се вижда на сайта:**
Датата се показва в декоративния медальон в ляво на картата на събитието — денят (число), месецът (на български) и годината се извличат автоматично от въведената дата.

```
╔════════╗
║   14   ║  ← ден
║  Юни   ║  ← месец (автоматично на български)
║  2025  ║  ← година
╚════════╝
```

> ⚠️ **Важно:** Форматът трябва да е точно `ГГГГ-ММ-ДД` с тирета. Пример: `2025-06-14`, не `14.06.2025` или `14/06/2025`.

---

### 2. Title (Заглавие)

| | |
|---|---|
| **Поле в Contentful** | `Title` |
| **Какво да въведете** | Името/заглавието на събитието |
| **Пример** | `Вечер на класическата музика` |

**Къде се вижда на сайта:**
Главното заглавие на картата на събитието — показва се с едър, декоративен шрифт в златист цвят.

---

### 3. Meta (Подзаглавие)

| | |
|---|---|
| **Поле в Contentful** | `Meta` |
| **Какво да въведете** | Кратко подзаглавие или слоган на събитието (1 изречение) |
| **Пример** | `Незабравими мелодии от епохата на романтизма` |

**Къде се вижда на сайта:**
Курсивният текст директно под заглавието — служи като кратко представяне преди пълното описание.

---

### 4. Description (Описание)

| | |
|---|---|
| **Поле в Contentful** | `Description` |
| **Какво да въведете** | Пълното описание на събитието |
| **Пример** | `Присъединете се към нас за вечер, изпълнена с музика на Бетовен, Шопен и Брамс. Концертът се провежда в Зала 1 на Националния дворец на културата.` |

**Къде се вижда на сайта:**
Основният параграф с информация за събитието. На обикновените карти се показват само първите 2 реда — целият текст се вижда на специалните (featured) карти.

---

### 5. Tag (Категория)

| | |
|---|---|
| **Поле в Contentful** | `Tag` |
| **Какво да въведете** | Категорията на събитието — една дума или кратка фраза |
| **Примери** | `Музика`, `Кино`, `Театър`, `Изложба`, `Лекция` |

**Къде се вижда на сайта:**
Малкият цветен етикет/бадж на картата на събитието. Текстът се изписва с главни букви автоматично.

---

### 6. Featured (Препоръчано)

| | |
|---|---|
| **Поле в Contentful** | `Featured` |
| **Какво да въведете** | Отметка (чекбокс) — включена или изключена |
| **По подразбиране** | Изключено (не е препоръчано) |

**Какво прави:**

| Стойност | Ефект върху сайта |
|---|---|
| **Изключено** (без отметка) | Събитието се показва като стандартна карта в списъка |
| **Включено** (с отметка) | Събитието се показва като голяма, акцентирана карта с пълно описание и по-ярко оформление |

> 💡 **Съвет:** Използвайте `Featured` само за специални или особено важни събития — за да се откроят от останалите.

---

## Обобщение

| Поле в Contentful | Показва се като | Задължително |
|---|---|---|
| `Date` | Дата в медальона (ден / месец / година) | Да |
| `Title` | Заглавие на картата | Да |
| `Meta` | Курсивно подзаглавие | Да |
| `Description` | Описание на събитието | Да |
| `Tag` | Цветен етикет с категория | Да |
| `Featured` | Специално оформление на картата | Не |
